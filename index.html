Mailspring

A desktop email client designed for macOS. Unified inbox, advanced search, snoozing, reminders, templates, and deep integration with Gmail labels and other IMAP providers. Clean interface with six themes, mail rules, and customizable shortcuts to achieve inbox zero.

Unified Inbox Across Accounts

Manage Gmail, Office 365, iCloud, Outlook.com, Yahoo, and any IMAP/SMTP provider from one streamlined view. Lightning-fast search across all accounts with Gmail-style queries.

Modern macOS Integration

Swipe to triage, Quick Look attachments, reply from Notification Center. Efficient sync respects App Nap for battery life without compromising freshness.

Pro Features Included

Read receipts, link tracking, send later, reminders, contact profiles, email analytics. Custom signatures with images, real-time translation, spell check, and easy RSVPs—all powered for any IMAP provider.

Advanced Tools

Offline support, undo send, themes including dark mode, gesture controls, and extensive localization in 9 languages plus 60 more. Reply-To composer field and bulk message export in version 1.21.0.

Enhanced Security

FAQ

Which email providers are supported?

All IMAP/SMTP including Gmail, Office 365, iCloud, Outlook.com, Yahoo. No Exchange ActiveSync.

Is this the Pro version?

Yes, this activated version includes all Pro features like read receipts, link tracking, and analytics—no subscription needed.

Does it work offline?

Full offline search and read support. Syncs efficiently when online.

Any macOS specifics?

Deep integration: gestures, Quick Look, notifications, App Nap. Optimized for Sonoma and later.

Updates and security?

Version 1.21.0 includes security audit fixes, HTTP/2 stability, and new features like Reply-To field.
